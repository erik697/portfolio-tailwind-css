Yuk Belajar Progamming

Saya berasal dari desa Nyitdah, kecamatan Kediri, kabupaten Tabanan. Masa kuliah adalah awal saya mulai mengenal bahasa pemrograman diawali dengan pembuatan flowchart kemudian pertama kali menggunakan bahasa C. Java adalah awal saya mengenal konsep OOP (Object Orientied Programming) dimana konsep itu masih saya pakai sampai sekarang. Bahasa yang paling saya suka adalah PHP dengan menggunakan framework laravel. selain bahasa yang saya sebutkan tadi saya juga belajar bahasa pemrograman Javascript, Phyton, dan Ruby
